# 0verVerification


## Contributing
### Install Dependencies
```
git clone https://github.com/0verseas/0verVerification.git
cd 0verVerification
npm install
```
### Setup
```
cp public/js/env.js.example public/js/env.js
```
edit the config file in `public/js/env.js`

### Testing
```
$ npm run serve
```

### Building
```
$ npm run build
```
